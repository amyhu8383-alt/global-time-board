# Global Time Board

Public standalone Next.js app for checking local time across key global markets.

## Route

- `/`

## Local preview

```bash
npm.cmd run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm.cmd run build
```

## Scope

This project only includes:

- Shenzhen base time
- global city time cards
- region and status filters
- search
- working status badges
- suggested contact actions

It does not include customer data, sales pipeline data, market calendar data, CRM integrations, or private business information.
