# Project Status

## 1. Project Name

Global Time Board

## 2. Project Purpose

A public, shareable global time board for checking local time across key global markets.

## 3. Project Scope

This project only shows:

- current local time
- local date
- timezone
- region
- working-time status
- suggested contact timing
- search and filters

## 4. What This Project Must Not Include

This project must not include:

- customer data
- customer follow-up assistant
- sales pipeline
- CRM
- boss report
- market calendar
- holiday calendar
- hurricane season tracking
- competitor intelligence
- private business data
- internal sales dashboard
- Amy B2B Sales Tools content

## 5. Current Route

- `/`

## 6. Current Features

- Shenzhen / China base time
- Time Window Summary
- Search by country, city, region, or timezone
- Region filters
- Status filters
- Global city cards
- Responsive card layout
- Working status badges
- Suggested action badges
- No private data stored

## 7. Working Status Logic

- Weekend: Saturday or Sunday
- Best Time Now: 09:00-11:30 local time, Monday-Friday
- Lunch Time: 12:00-13:30 local time, Monday-Friday
- Good Time: 14:00-17:30 local time, Monday-Friday
- After Hours: before 09:00 or after 18:00 local time, Monday-Friday
